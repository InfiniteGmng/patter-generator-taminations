
# 2/3 Recycle

From a wave only: Do the first two parts of [Recycle](../ms/recycle.md), that is,
[Centers Fold](../ms/fold.md) (and all adjust to a Box Circulate formation), 
then [Box Counter Rotate 1/4](../a2/box_counter_rotate.md).
